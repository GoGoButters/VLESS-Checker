How to deploy a new Node

Nodes are headless Docker containers that pull proxy lists from this master panel, check them locally using Sing-Box, and report the results back. This allows you to build regional proxy lists and bypass DPI blocks from different IPs.

# 1. Clone the repository on your remote worker server:
git clone https://github.com/GoGoButters/VLESS-Checker.git
cd VLESS-Checker/node

# 2. Add your node token and master URL to docker-compose.yml:
nano docker-compose.yml
# Set MASTER_URL=https://YOUR_PANEL_DOMAIN
# Set NODE_TOKEN={{ settings.node_api_token if settings else 'NOT_SET' }}

# 3. Build and run the node container
docker compose up --build -d

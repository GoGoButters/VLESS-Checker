# VLESS Proxy Checker

VLESS Proxy Checker — это автономный сервис с веб-интерфейсом для агрегации, проверки и автоматической раздачи рабочих VLESS конфигураций (Reality/TLS). 
Проект идеально подходит для автоматического тестирования огромных списков подписок с прокси-серверами — приложение само проверит пинг, соединения, отбросит нерабочие серверы и выдаст вам только лучшие из лучших в формате автообновляемой подписки.

## Особенности
- **Многопоточное тестирование**: Проверка серверов встроенным ядром Xray.
- **Двухэтапная проверка**: TCP пинг + строгая проверка через реальные сайты (генерируемый 204 статус, проверка тела ответа от google.com/youtube.com).
- **Динамические проверки**: Возможность настройки через UI любых ссылок для проверок. Прокси ранжируются по количеству стабильно пройденных тестов и минимальному пингу.
- **Встроенный планировщик**: Настройте интервал от 15 минут до 24 часов, и система будет проверять и обновлять прокси в фоне.
- **Webhook раздача**: Итоговые проверенные списки прокси доступны по секретной ссылке, где у всех названий (remark) заменены имена на порядковые номера (`#1`, `#2`...) по рейтингу качества.
- **Стильный UI**: Панель управления (Admin-панель) в современном тёмном Glassmorphism дизайне.

## 🚀 Установка на чистый VPS

Проект спроектирован для мгновенного развертывания с использованием Docker. Вам потребуется только `docker` и `docker-compose` (либо плагин `docker compose`).

### 1. Подготовка сервера

Подключитесь к вашему VPS и обновите систему, а затем установите Docker:

```bash
# Обновляем пакеты
apt update && apt upgrade -y

# Устанавливаем curl и git
apt install curl git -y

# Устанавливаем Docker (официальный скрипт)
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh
```

### 2. Загрузка проекта

Склонируйте репозиторий в удобную папку (например, `/opt/vpn-checker`):

```bash
git clone https://github.com/GoGoButters/VLESS-Checker.git /opt/vpn-checker
cd /opt/vpn-checker
```

### 3. Запуск через Docker Compose

Внутри папки репозитория уже лежит готовый конфигурационный файл `docker-compose.yml`. Все данные (настройки, база SQLite, результаты проверок) будут сохранены в защищенном Docker volume.

Просто выполните команду для старта:

```bash
docker compose up -d --build
```

Приложение скачает все зависимости, настроит ядро Xray-core внутри контейнера и запустит веб-интерфейс.

## ⚙️ Настройка приложения

После успешного запуска панель управления будет доступна по адресу вашего сервера на порту `8000`:
* 👉 **http://<IP_ВАШЕГО_СЕРВЕРА>:8000**

**Данные по умолчанию для входа:**
* **Пароль:** `admin`

*(Обязательно смените пароль при первом входе в разделе "Settings")*

### Как начать пользоваться:
1. Зайдите во вкладку **Subscriptions** и нажмите "Add Subscription". Вставьте URL подписки со списком VLESS ссылок.
2. Во вкладке **Test URLs** убедитесь, что сайты генерации 204 статуса и Google/YouTube добавлены (они добавляются по умолчанию).
3. Во вкладке **Settings** задайте:
   - *Schedule Interval (minutes)*: например, 60 для автопроверки раз в час.
   - *Max Proxies in Webhook*: например, 10 (сервер выдаст 10 лучших).
4. Вернитесь в **Dashboard** и нажмите **"Run Full Test"**. Дождитесь окончания прогресс-бара.
5. Во вкладке **Proxies** появятся проверенные рабочие сертификаты.
6. Выдайте своим пользователям (или вставьте в VPN клиент) вебхук-ссылку из низа Дашборда: `http://<IP>:8000/secret-distrib`

## 🔨 Обновление 

Если выйдет новая версия проекта, для обновления достаточно зайти в директорию и перезапустить контейнер:

```bash
cd /opt/vpn-checker
git pull
docker compose up -d --build
```
